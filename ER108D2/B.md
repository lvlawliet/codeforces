## 题目翻译

**右侧最大值**

给定一个长度为 n 的整数数组。

不断执行以下操作直到数组为空：
1. 找到数组中的**最大值**（若有多个最大值，选**最右边**那个）
2. 将该元素及其**右边所有元素**一并删除

求总共执行了多少次操作。

---

## 解题思路

### 关键观察

每次操作都会找到"当前数组的最右最大值"并截断。  
注意到：**每次操作后，剩余数组是原数组的某个前缀**。

所以问题等价于：**在原数组中，从右往左，能找到多少个"前缀最大值位置"？**

更具体地说，一个位置 i 会成为某次操作的截断点，当且仅当：

> `a[i]` 是 `a[0..i]` 中的最大值（即从左到右的**前缀最大值**）

### 证明思路

- 第一次操作：找整个数组最右最大值 → 必然是全局最大值的最右位置
- 第二次操作：在剩余前缀中找最右最大值 → 是剩余部分的前缀最大值
- 每个前缀最大值恰好对应一次操作

因此，**答案 = 原数组从左到右前缀最大值的个数**。

### 示例验证

```
数组: 3 1 2 3 2
前缀最大值位置: 
  i=0: a[0]=3, 最大 ✓
  i=1: a[1]=1, 不是最大 ✗
  i=2: a[2]=2, 不是最大 ✗
  i=3: a[3]=3, 与前缀最大相等（取最右）✓
  i=4: a[4]=2, 不是最大 ✗
答案 = 2
```

## C++ 代码

```cpp
#include <bits/stdc++.h>
using namespace std;

int main()
{
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int t;
    cin >> t;

    while (t--)
    {
        int n;
        cin >> n;
        vector<int> a(n);
        for (int i = 0; i < n; i++)
            cin >> a[i];

        int ans = 0;
        int curMax = 0;
        for (int i = 0; i < n; i++)
        {
            if (a[i] >= curMax)
            {
                curMax = a[i];
                ans++;
            }
        }

        cout << ans << "\n";
    }

    return 0;
}
```
